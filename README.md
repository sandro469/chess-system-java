# Chess System Java ♟️

Projeto desenvolvido em Java simulando um sistema de xadrez em console.

## Objetivo do Projeto

Desenvolver uma aplicação aplicando conceitos de Programação Orientada a Objetos,
criando uma estrutura organizada para representar peças, tabuleiro e regras do jogo.

## Tecnologias Utilizadas

- Java
- Eclipse IDE
- Git/GitHub

## Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Herança
- Polimorfismo
- Classes abstratas
- Composição de objetos
- Tratamento de exceções
- Organização de pacotes

## Funcionalidades

- Movimentação das peças
- Validação de movimentos
- Captura de peças
- Controle de turnos
- Xeque
- Xeque-mate
- Promoção de peões

## Aprendizados

Projeto desenvolvido para praticar lógica de programação,
modelagem de objetos e construção de regras de negócio utilizando Java.
